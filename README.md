-Hướng dẫn chạy project
Để khởi chạy dự án này trên máy tính cá nhân hoặc thiết bị di động, hãy làm theo các bước sau:

Yêu cầu hệ thống:
Đã cài đặt Node.js (phiên bản LTS).

Đã cài đặt Expo CLI (npm install -g expo-cli).

Ứng dụng Expo Go trên điện thoại (Android/iOS) hoặc trình duyệt web.

Các bước thực hiện:
Cài đặt thư viện:
Mở terminal tại thư mục gốc của project và chạy lệnh:

Bash
npm install
Cài đặt các bộ icon cần thiết:

Bash
npx expo install @expo/vector-icons expo-status-bar
Khởi chạy dự án:

Bash
npx expo start
Xem kết quả:

Quét mã QR bằng ứng dụng Expo Go để xem trên điện thoại.

Nhấn phím w để xem trên trình duyệt web (đã được cấu hình bo góc 414px để khớp giao diện mobile).
